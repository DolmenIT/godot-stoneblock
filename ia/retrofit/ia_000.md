### Update

```typescript
update(time: number, delta: number): void {
    super.update(time, delta);
    
    // Mettre à jour travelling
    if (this.cameraTravelling) this.cameraTravelling.update(delta);
    
    // Faire suivre la lumière de caméra
    if (this.cameraLight && this.camera) {
        this.cameraLight.setPosition(
            this.camera.position.x,
            this.camera.position.y,
            this.camera.position.z
        );
    }
    
    // Animations dynamiques
    this.updateDynamicAnimations(delta);
    
    // Mettre à jour managers et modèles
    if (this.lightingManager) this.lightingManager.update(delta);
    if (this.logoDagx3D) this.logoDagx3D.update(delta);
    if (this.logoStoneBlock3D) this.logoStoneBlock3D.update(delta);
}
```

## Méthodes spécifiques

### hideStoneBlockLogo()

```typescript
private hideStoneBlockLogo(): void {
    if (!this.logoStoneBlock3D) return;
    
    const model = this.logoStoneBlock3D;
    const startTime = Date.now();
    const duration = 1000;
    
    const animateOpacity = () => {
        const elapsed = Date.now() - startTime;
        const progress = Math.min(elapsed / duration, 1.0);
        const opacity = 1.0 - progress;
        
        model.setOpacity(opacity);
        
        if (progress < 1.0) {
            requestAnimationFrame(animateOpacity);
        }
    };
    
    animateOpacity();
}
```

**Durée :** 1000ms (fade out complet)

### fadeToWhite()

```typescript
private fadeToWhite(): void {
    console.log('🔍 Fading to white');
    this.sceneFG.fadeToWhite(1500, 0xeeeeee).then(() => {
        console.log('✅ Fade to white terminé');
        this.fadeToWhiteCompleted = true;
    });
}
```

**Configuration :**
- Durée : `1500ms`
- Couleur : `0xeeeeee` (gris clair, pas blanc pur)
- Flag : `fadeToWhiteCompleted` pour tracker l'état

### loadWelcomeScene()

```typescript
private loadWelcomeScene(): void {
    if (this.fadeToWhiteCompleted) {
        (document as any).previousSceneFG = this.sceneFG;
        document.game.loadScene('200_welcome_scene');
    } else {
        setTimeout(() => {
            this.loadWelcomeScene();
        }, 100);  // Vérifier toutes les 100ms
    }
}
```

**Logique :**
- Attend que `fadeToWhiteCompleted` soit `true`
- Stocke `previousSceneFG` pour le nettoyage
- Charge la scène `200_welcome_scene`

### updateDynamicAnimations()

```typescript
private updateDynamicAnimations(delta: number): void {
    // Rotation du logo StoneBlock sur 3 axes
    const stoneblockMesh = this.logoStoneBlock3D?.getMainObject();
    if (stoneblockMesh) {
        stoneblockMesh.rotation.x += this.stoneblockRotationSpeed.x * delta;
        stoneblockMesh.rotation.y += this.stoneblockRotationSpeed.y * delta;
        stoneblockMesh.rotation.z += this.stoneblockRotationSpeed.z * delta;
    }
}
```

**Vitesses :**
- X : `0.3 * delta`
- Y : `0.5 * delta`
- Z : `0.2 * delta`

## Points clés pour migration

1. **Travelling :** Séquence avec pauses et callbacks → Godot `Tween` + `Curve3D` ou `Path3D`
2. **Bloom sélectif :** Layers Three.js → Godot Rendering Layers (implémentation custom nécessaire)
3. **Fade :** `sceneFG` Three.js → `CanvasLayer` + `ColorRect` dans Godot
4. **Synchronisation UI :** Appels directs → Signaux Godot ou appels de méthodes
5. **Rotation StoneBlock :** Update chaque frame → `_process(delta)` dans Godot
6. **Starfield :** `CustomStarfield3D` → Script custom Godot avec `Sprite3D`
