# 🖼️ SB_UI_Layer

Conteneur spécialisé (`CanvasLayer`) pour regrouper les éléments d'interface StoneBlock et gérer leur affichage global.

## ⚙️ Paramètres

### Layer Settings
- **Is UI Visible** : Permet de masquer/afficher toute l'interface de ce calque en un clic.
- **Layer** (Hérité) : Définit la priorité d'affichage (0 par défaut, mis à 10 par le script à l'initialisation).

## 🛠️ Utilisation
Utilisez ce nœud comme parent de vos composants `SB_DebugConsole` ou `SB_Button` pour organiser votre interface de manière propre et centralisée.
