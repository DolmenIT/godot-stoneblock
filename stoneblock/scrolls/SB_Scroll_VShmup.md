# 🚀 SB_Scroll_VShmup
Composant de défilement pour les éléments de décor ou de fond dans le mode SHMUP.

## Fonctionnalités
- Déplacement constant dans une direction donnée.
- Option de bouclage infini (`use_infinite_scrolling`) pour les textures ou modèles répétables.
- Ajustement de la vitesse de défilement via `scroll_speed`.

## Utilisation
1. Attacher ce script à un nœud de décor (ex: un plan d'eau, un champ d'étoiles).
2. Configurer la direction de défilement (généralement `0, 0, 1` pour le bas).
3. Activer le bouclage infini si l'objet doit se répéter indéfiniment.
